Lists Usage
Unordered List An unordered list is used by default with a bullet to present a list of items in an unspecific order. On a webpage unordered list should be used to describe a list of general items. The tag used to start an unordered list is <ul>
Ordered List An ordered list is used by default with numbers to present a list of items in a specific order. On a webpage an ordered list should be used to dictate a series of steps needed to complete a task in a specific order. The tag used to start an ordered list is <ol>
Definition List A definition list is used to define terms and provide descriptions. The tag used to open a definition list is <dl>
HTML
HTML is an acronym for Hyper Text Markup Language. It is a code used to structure a web page and its content.
CSS
CSS is an acronym for Cascading Style Sheet. It is a declarative language used to control the visual aspect of webpages when in a browser format.
FTP
FTP is an acronym for File Transfer Protocol. It is a standard communication protocol used to transfer computer files from a server to a computer network.
SSH
SSH is an acronym for Secure Shell. It is a network communication protocol used to transfer hypertext and share data.
